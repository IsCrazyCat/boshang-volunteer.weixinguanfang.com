常见问题解答
  • 注册问题
    • 问

      港澳台用户如何注册志愿者账号？

      港澳台用户自行在平台申请注册志愿者账号即可。（需要邀请证明人作为证明，然后再由组织管理员进行身份校验信息进行审核（在实名制校验中））

    • 问

      登录提示“您的账号正在实名审批中，请耐心等待”。

      由于帐号在等待公安系统的身份证校验，志愿者注册提交资料同时，系统将会志愿者身份信息进行实时校验。登录使用账号。正常情况下，志愿者身份信息校验可实现秒过。

    • 问

      关于身份信息校验情况说明？（新注册用户一般为24小时即可处理完成，老用户的需要进行排队审核。）

      身份信息校验是我系统和公安系统对接的身份验证过程，主要为校验姓名和身份证号是否一致。

    • 问

      注册时发现身份证信息已存在，但是不清楚帐号填写的手机和邮箱？

      可能由于学校或单位采用excel表格批量导入信息所致。 （1）先尝试使用身份证号作为帐号，6个0作为密码进行登录； （2）可通过本系统“忘记密码”，在输入登录账号时尝试用手机号/身份证号作为账号，经手机号码和邮箱获取验证码找回密码。 找回密码申诉。如还不解决问题将个人身份证+姓名+手机号+问题发送至kf@izyz.org 客服邮箱处理。 步骤：“本系统登录”—“忘记密码”—“输入身份证号或者手机号码为主”—“选择第三个申诉入口” （如果清楚自己的注册组织的用户最好联系自己归属组织的管理员进行密码重置）。

    • 问

      在一号通注册后,在本系统登录账号显示密码错误，找回密码时输入手机号码或者身份证号码显示账号不存在？

      由于在一号通注册账号但没有进行完善资料。 步骤:“一号通登录”—“忘记密码”—“找回密码”—“进行完善资料”。

    • 问

      同一个身份证注册了志愿者还能注册成为组织管理员吗?

      能。注册志愿者，也可以注册管理员。 （注意：在注册管理员时的登录账号必须设置与个人登录账号不一样，管理员帐号是不可以参加志愿活动）。

    • 问

      归属组织选错了需要更改组织？

      进入用户中心-完善资料，选择正确的归属组织，最多支持加入5个组织。（需要联系您所选择归属组织的管理员进行审核）。

    • 问

      注册组织账号如何区分已备案和未备案？

      已备案指的是在民政部门登记成立的法人组织。 未备案指的是未在民政部门登记备案的社会组织或其他。 （选择其他，可无需填组织机构代码或者统一社会信用代码） 注；在民政部门登记成立的法人组织将由登记备案所在的民政管理员进行审核。未在民政部门登记备案的社会组织或其他组织将由归属组织的管理员进行审核。

    • 问

      登录账号显示身份校验不通过？

      核实姓名或者身份证号是否正确，信息有误请修正重新校验信息。（姓名或者身份证号无误由管理员进行“重新校验”，如“重新校验”无法通过，请提供身份证手持图片发到kf@izyz.org 。）

    • 问

      注册成功可以登录账号了，可显示归属组织还在审核中？

      新注册的用户只要身份信息校验通过即可登录使用账号，无需等待注册时所选组织的审核通过。

    • 问

      关于注册时（或者是忘记密码），手机（或者邮箱）获取不了验证，如何解决？

      （1）可查询手机是否开通短信智能拦截或者是否安装的安全软件自动拦截。 （2）可查询邮箱设置是否开通拒收邮件功能或者进入垃圾邮件中查询。 （3）在拦截处查询不到短信请向i志愿客服反应，并提供接收验证码信息的手机号码或邮箱；或者身份证+姓名+手机（邮箱）+问题发到客服邮箱kf@izyz.org 。

    • 问

      登录组织/团体账号出现账号密码错误或者用户不存在情况，点击“忘记密码”，进行密码修改。

      请在第一步输入帐号时尽量输入登录账号（手机号/身份证号）作为账号，通过短信、邮箱、提交申诉资料三种方式找回密码。如还不解决问题将个人身份证+姓名+手机号+问题发送至kf@izyz.org 客服邮箱处理。

  • 活动问题
    • 问

      关于面验，是怎么操作的？

      第一次参加志愿者活动时需要面验才可以进行扫码签到。 面验：面验指的是活动管理员查看需要面验的人员的身份证件，以确保信息真实性。 注意以下几点： 1.活动考勤前需要对未面验的志愿者进行面验，如未面验，会造成考勤失败。 2.注册信息与身份证信息一致，面验通过。注册信息与身份证信息不一致，面验不通过。 3.平台在活动开始前会以短信方式通知到需要面验志愿者携带身份证参加活动。 4.面验通过的志愿者，下次活动无需面验。 5.面验操作可在微信城市服务端、支付宝城市服务端、pc端进行操作。 步骤：PC端“活动管理”—“活动查询”—“搜索活动名称”—“操作”—“考勤”—“面验” 微信端 登录管理员帐号“我的”—“活动管理”—“活动考勤”—“搜索活动名称”—“面验”

    • 问

      如何邀请录用？

      邀请是指邀请组织名下的志愿者进行参加活动，以站内信的方式发给用户报名活动。 录用是指用户报名活动，管理员在后台进行录用志愿者或者拒绝志愿者。 步骤：PC端 “活动管理”—“邀请录用”—“搜索活动名称”—“操作”—“邀请/录用” 微信端：登录管理员账号“我的”—“活动管理”—“邀请录用”—“搜索活动名称”—“邀请/录用”

    • 问

      管理员发布活动，但在PC端或者微信端搜索不出活动？

      是由于这场活动时间超过8小时，需要上级管理员进行审核。 （每天考勤时长不超过8小时，如果超出8小时请勾选申请，上级审核通过后，以实际时长为准，但也不超过11小时。）

    • 问

      如何查询这场活动是否超过8小时

      步骤：PC端 “活动管理”—“活动审核” —“搜索活动名称”—“操作”—“审核” 微信端：登录管理员帐号“我的”—“活动管理”—“活动审核”—“搜索活动名称”—“审核”

    • 问

      培训活动是否加入服务时长？

      不可以。发布活动时，请注意选择活动性质，现在新增培训活动，由此产生的时长将录入到培训时长中。（查看培训时长需要在PC端登录账号。）

    • 问

      用户在微信端报名活动时显示未登录或登录超时？

      请按以下步骤：微信端登录志愿者账号—“我的”—公众号解绑—退出账号—清除缓存—重新登录账号

    • 问

      管理员如何发布活动？

      （1）发布活动时，请注意选择活动性质，现在新增培训活动，由此产生的时长将录入到培训时长中。 （2）发布活动时，需要选择服务地点坐标和考勤地理范围。（服务地点坐标以活动地点为准，关系到扫码考勤是否成功，请知悉。在指定理范围才可以进行扫码签到。） （3）管理员在发布活动时，可选择自定义报名，点击后可自行添加报名条件，志愿者在报名时必须输入相应字段才能报名。 （4）发布活动时，每天考勤时长不超过8小时，如果超出8小时请勾选申请，上级审核通过后，以实际时长为准，但也不超过11小时。 （5）活动发布时，活动宣传照片、活动地址为必填。

    • 问

      发布活动，邀请、录用、考勤全部程序走完，志愿时什么时候会显示？是不是需要公示？等公示期过了时长才录入到个人信息中？

      不是，考勤结束后 服务时长隔天才会更新

    • 问

      用补录方式给服务时数时弄的，志愿者多给了服务时数、导致时长比正常多出，管理员是否撤回操作的吗？

      是不可以的，没有取消考勤的功能

    • 问

      活动进行中或者结束的情形下,管理员进行撤销活动,这个活动志愿者的志愿时会消失吗？还是会照常记录

      活动时长是存在的。

    • 问

      志愿者活动考勤时，只签到忘了签退，怎么办？

      (1)签到和签退时间会产生一个时间差；时间差就是志愿者的服务时长，忘了签退算是考勤失败，没有服务时长。 (2)在已签到没有签退的情况下，由于已签到是存在考勤，人工补录时长当天补录不了时长，可隔天由管理员进行补录操作。

    • 问

      什么是历史活动补录？

      对于以前已经开展、且未发发布过的活动，管理员可通过历史活动补录完成补录，4.0中“发布活动”将不能再发布往期的活动。 步骤：“活动管理”—“发布历史活动”—“录用”—“添加考勤”—“提交材料” “提交资料”后等待上级管理员进行审核即可。

    • 问

      如何区别3种记录服务时长？

      （1）由活动发布人提供签到签退二维码给志愿者进行签到签退 。（每天可以签到签退3次） 步骤：微信端登录管理员帐号“我的”—“活动管理”—“活动考勤”—“搜索活动名称”—“签到/签退按钮”—“生成二维码” （2）扫志愿者证二维码记录服务时长 步骤：微信端登录管理员帐号“我的”—“活动管理”—“活动考勤”—“搜索活动名称”—“签到/签退按钮”—“志愿者证签到”—“管理员进行扫志愿者证二维码” （3）自行签到签退 注意如下：（1）活动参与人数多余50人，才可以选择自行签到签退 （2）发布活动时考勤方式选择自行签到签退，选择考勤地理范围 步骤：微信端 登录志愿者帐号“我的”—“我的活动”—“参与中”—“自行签到” 活动结束后，需要管理员进行操作 微信端 微信端登录管理员帐号“我的”—“活动管理”—“行签到签退确认”—“搜索活动名称”—“考勤确认” PC端 “活动管理”—“行签到签退确认”—“搜索活动名称”—“操作”—“考勤确认

    • 问

      用户对活动或者时长有异议该怎么办？（包括发布补录活动）

      举报可以是针对活动的举报，也可以是对志愿者时数录入的举报。 （1）活动举报：在活动详情页面，活动标题右下角处，点击举报，按要求提交材料，相关举报将直接反馈至省级管理员。 （2）志愿者时长举报：在时长公式页面，选择活动查看，可以看到每位参加活动的志愿者通过补录录入的时数和通过扫码录入的时数。若有异议可以点击“我有异议”进行举报，信息将直接反馈省管理员。

    • 问

      用户参加志愿者活动颁布过5星志愿者证书，该如何补录时长？

      可以通过举证补录进行补录。 举证补录仅针对高级管理员以上权限的管理员，针对以往颁发过星级志愿者服务时长认定的证书，进行审核后通过后，在系统中完成举证审核补录功能。 步骤：“登录管理员帐号”—“活动管理”—“举证材料补录”—“添加补录”

    • 问

      是否活动结束后系统自动将活动结项？

      新系统上线之后，活动是不需要结项，到了活动日期截止日期会系统自动转为结束状态； 现在平台也取消了活动结项功能，现活动根据活动发布时候填写的日期自动执行、自动结束。在活动执行中和活动结束后方可执行考勤操作

    • 问

      目前志愿服务时数是如何补录？

      分为3种（1）实时活动补录 （2）历史活动补录（3）举证补录

    • 问

      用户参加某场活动显示“只允许本组织成员报名”？

      说明这场活动只有本组织名下的成员才可以报名。（不包括下级组织）

  • 组织团体问题
    • 问

      管理员登录账号时显示密码错误？

      在登录账号与密码正确情况下，请核实是否选择登录入口有误。

    • 问

      如何变更组织团体管理员？

      步骤：登录管理账号—组织管理— 管理员变更 （如没有运营者帐号需添加运营者帐号再进行操作）

      组织账号更改归属组织是否需要审核？

      需要审核，联系所选择的归属组织的管理员进行审核。

    • 问

      用户注册组织账号时选择已备案是选择谁审核？

      选择已备案是由民政局进行审核的。（需要联系您选择民政局进行审核）

  • 志愿者证问题
    • 问

      申请办理志愿者证已经半个月了，怎么还没有收到呢？

      志愿者证上需印制志愿者的基本信息和二维码，要特殊制卡，不像在柜台即办即可领卡，制卡时间大概为2~3个月，请耐心等待办卡。如需了解办卡进度可登录i志愿个人中心的志愿者证申请进度查看，或者电话银行客服咨询。 中国银行：信用卡 客服热线400 669 5566邮箱gdzhzyz@163.com 借记卡客服热线95566客服邮箱boczyz@sina.com 光大银行：信用卡 客服热线400 788 8888  借记卡客服热线 95595。

    • 问

      申请办理志愿者证怎么收到银行的其他信用卡呢？

      在申请办理信用卡类型的志愿者证时，您选择了办理其他信用卡，导致申请志愿者证失败（注意申请时不要选择办理其他信用卡）。需要您重新登录i志愿个人中心申请办理志愿者证即可。

    • 问

      我已经办理志愿者证为什么收不到志愿者服装呢？

      服装由银行赠送的，需要您使用志愿者证完成首笔消费后，银行才会根据你申请志愿者证时选择的尺码开始制作服装，因要提取制作信息以及实名印制。故制作时间周期较长，一般是首刷后2个月,服装制作配送进度可在i志愿个人中心查看。详细可以咨询银行客服。如您一直未使用志愿者证进行首笔消费，是收不到志愿者服装的。

    • 问

      在网站上已经查询到制卡完成，怎么还没有收到卡呢？

      制卡完成后，银行需要分发到各领取网点，领取志愿者证以银行通知短信为准。如有其他问题可咨询银行客服查询， 中国银行：信用卡 客服热线400 669 5566邮箱gdzhzyz@163.com 借记卡客服热线95566邮箱boczyz@sina.com 光大银行：信用卡 客服热线400 788 8888  借记卡客服热线 95595。

    • 问

      志愿者证丢失或者更换新卡是否需要收取工本费？

      需要收取工本费， 需本人前往银行网点办理换卡。

    • 问

      用户参加志愿活动中发生事故该如何进行保险理赔？

      志愿者申请是（中行或者光大)志愿者证，请通过i志愿平台（帮助）搜索注册志愿者证理赔文件公示（中行卡或者光大）进行下载文件进行填写资料。

    • 问

      志愿者证审批中显示相片获取不到，是否需要重新上传？

      需要重新上传图片，请登录i志愿电脑端在志愿者证查询进度那里上传图片。（注意上传证件图片的规格尺寸:证件头像为注册志愿者证正面的卡面头像，请按照证件的标准规则尺寸进行上传。照片宽与高的比例为1:1.25，宽度不低于360px，高度不低于450px，底色为白色。）

  • 其他问题
    • 问

      如何用户名设置？

      志愿者在pc端登录个人账号后，在“完善资料”——“个人资料”中，按规定填写并提交保存即可。 注意：用户名只能设置一次，设置后用户名可作为登录账号登录。

    • 问

      如何查看我的消息？

      志愿者登录个人账号后，在“我的消息”栏目中，即可查看目前系统、组织、活动类型的消息，如有站内私信，将出现在@与我相关中。 管理员账号是登录账号在个人中心进行查看消息。

    • 问

      使用多种方式登录

      志愿者登录支持微信扫码、身份证号、用户名、手机号码、绑定邮箱等五种方式登录。

    • 问

      活动报名新增自定义条件报名。

      管理员在发布活动时，可选择自定义报名，点击后可自行添加报名条件，志愿者在报名时必须输入相应字段才能报名。

      手机获取验证码提示已超5次怎么办？

      关于登录账号情况说明？

      （1）每个用户有自身的一个登录帐号，登录帐号可分为手机号码，身份证号、邮箱或者注册时自设的用户名等等情况。 （2）微信端与电脑端是相统一的，登录帐号+密码方可登录；系统暂时是不支持更改登录帐号的。 如志愿者是本组织下的志愿者，可在后台查看一下登录帐号，告知志愿者的登录帐号，如不是本组织的话，用Excel表格统计将身份证+姓名+手机号码+问题发到客服邮箱。kf@izyz.org

      邀请志愿者，怎么没有收到短信的呢？

      （1）邀请通知是以微信通知、短信通知、站内信三种方式。志愿者账号关注并绑定i志愿服务号邀请通知以微信通知和站内信方式推送，如果没有的话将以短信通知推送。注意：如果志愿者账号绑定了公众号却未关注i志愿服务号，将导致邀请通知推送失败，请检查是否关注了i志愿服务号。 微信公众号绑定方法：关注i志愿微信服务号，点击志愿者菜单的绑定账号，微信自动推送一条绑定推送，点击该链接根据提示绑定即可。
